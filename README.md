# Transfer

> Encrypted file transfer utility

> Built with `rust` and `vue.js`.


- Server (base project dir):
    - Nightly `rust` is currently required for `rocket`
    - `rustup override set nightly`
    - `cargo run -- serve --log`
- Webapp (`/web`):
    - `npm run dev`
    - open `http://localhost:8080`
